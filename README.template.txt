每日发货看板 - 使用说明

1. 启动软件
   双击 fa-huo-dashboard.exe 打开看板。

2. 第一次使用
   点击顶部的“数据目录”，选择一个固定目录保存本地数据，例如 D:\发货看板数据。
   然后点击“选择文件”或把 Excel 拖进窗口，软件会读取“发货历史记录”和“发货明细”。

3. 日常使用
   选择统计日期后，软件会优先读取本机历史库。
   如有新 Excel，重新导入即可更新历史库。
   如需按仓库材料类型识别客户发货大类，点击“材料类型表”，选择从金山文档导出的产品进出存 Excel。

4. 导出
   点击“导出图片日报”“导出 CSV”或“导出数据”后，可以选择保存位置。
   点击“保存日报包”会保存到当前数据目录下的 reports 文件夹。

5. 备份和恢复
   点击顶部“备份”可生成 zip 备份包。
   点击“恢复”可从备份包恢复历史数据库，恢复会覆盖当前数据目录里的历史数据。

6. 拷贝到其他电脑
   直接拷贝整个 fa-huo-dashboard 文件夹。
   到新电脑后先打开软件，再用“恢复”导入原电脑生成的备份包。

主要文件说明：
- data\history.sqlite：历史数据库
- reports\：日报包和备份默认保存目录
- app_settings.json：桌面设置
- shipment_config.json：业务规则配置
